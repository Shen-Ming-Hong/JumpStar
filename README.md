# 跳跳星 (JumpStar)

## 專案簡介

跳跳星是一個基於 Arduino UNO 的互動式跳躍遊戲裝置，透過紅外線感測器偵測玩家跳躍高度，並使用 WS2812B 光條呈現視覺回饋效果。

## 硬體需求

-   **主控板**: Arduino UNO
-   **光條**: WS2812B LED 燈條
-   **感測器**: 紅外線感測器（數位訊號）

## 功能特色

-   即時偵測玩家跳躍動作
-   根據玩家離地時間計算跳躍高度
-   依據跳躍高度動態變化燈光效果
-   提供視覺化的跳躍回饋

## 工作原理

1. 紅外線感測器持續監測玩家位置
2. 當玩家離開地面時開始計時
3. 根據離地時間推算跳躍高度
4. WS2812B 燈條顯示對應的燈光效果
